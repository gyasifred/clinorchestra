### Embedding Strategy:

1. **Document Chunking**: Split large guidelines (>100 pages) into sections by diagnosis/topic
2. **Metadata Tagging**: Tag each chunk with:
   - Diagnosis ICD codes
   - Clinical category (cardiovascular, infectious, etc.)
   - Guideline type (diagnostic criteria, treatment, risk stratification)
   - Publication year
   - Strength of recommendation (when available)
3. **Semantic Search Optimization**: Include:
   - Clinical synonyms (e.g., "MI" = "myocardial infarction" = "heart attack")
   - ICD code mappings
   - Medication names and classes
   - Laboratory test names and abbreviations

## CITATION FORMAT FOR ANNOTATIONS

When using these guidelines in clinical annotations, cite as:

**Format**: "Per [Society] [Year] guidelines, ..."

**Examples**:
- "Per Surviving Sepsis Campaign 2021 guidelines, this patient meets sepsis-3 criteria with SOFA score ≥2"
- "Per ACC/AHA 2023 ACS guidelines, troponin elevation meets criteria for NSTEMI"
- "Per KDIGO 2012 staging, this represents Stage 3 AKI based on creatinine increase"
- "Per DSM-5 criteria, patient meets 7/9 symptoms for major depressive disorder"

## KEEPING RESOURCES UPDATED

### Guideline Update Schedule:
- **Cardiovascular**: ACC/AHA updates every 3-5 years
- **Infectious Disease**: IDSA updates as new evidence emerges
- **Kidney**: KDIGO major updates every 5-10 years
- **Psychiatric**: APA updates every 5-10 years, DSM less frequently
- **Oncology**: NCCN updates continuously (check quarterly)

### How to Monitor Updates:
1. Subscribe to professional society newsletters
2. Set PubMed alerts for "[condition] AND guideline"
3. Check guideline websites annually
4. Monitor high-impact journals (NEJM, JAMA, Lancet, Circulation, JACC)
