This document provides links to evidence-based clinical guidelines, diagnostic criteria, and key resources for each of the top 20 diagnoses.

## How to Use These Resources

### For Annotation (Task 1):
1. Review relevant guidelines for the diagnosis
2. Incorporate diagnostic criteria into your annotation
3. Use specific terminology from guidelines
4. Reference scoring systems where applicable

### For Classification (Task 2):
1. Use diagnostic criteria to assess likelihood
2. Apply clinical scores to quantify severity
3. Cross-reference multiple conditions
4. Consider differential diagnosis as per guidelines

### For Creating Extras:
1. Extract key diagnostic criteria from guidelines
2. Summarize severity scores and staging
3. Note red flags and critical findings
