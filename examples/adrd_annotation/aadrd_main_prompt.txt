**TASK:** Three-step diagnostic formulation to classify ADRD vs Non-ADRD.

**STEP 1 - SEVERITY:** No impairment / MCI / Mild / Moderate / Severe dementia
- KEY: Functional impairment distinguishes MCI from dementia
- MCI = cognitive impairment + functional independence PRESERVED
- Dementia = cognitive impairment + functional IMPAIRMENT

**STEP 2 - SYNDROME:** Pattern of cognitive deficits
- Domains: memory, executive, visuospatial, language, behavior
- Dementia requires >=2 domains affected
- Patterns: amnestic, executive, language, visuospatial, behavioral, mixed

**STEP 3 - ETIOLOGY:** ADRD vs Non-ADRD
- ADRD: AD, VaD, DLB, FTD, mixed
- Non-ADRD: Delirium, psychiatric, medical, medications
